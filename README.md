# 屏幕取色工具
一款基于浏览器原生 `EyeDropper API` 开发的屏幕取色工具，采用日式浮世绘经典配色风格，支持PC/移动端取色、颜色值复制，纯前端实现无需后端依赖。

## ✨ 功能特点
1. **核心取色**：点击按钮唤起系统级取色器，精准捕获屏幕任意像素的十六进制颜色值（`#xxxxxx`）；
2. **视觉展示**：实时预览取色结果，直观显示颜色块与对应的十六进制代码；
3. **便捷复制**：一键复制颜色值到剪贴板，复制成功有明确提示；
4. **跨端适配**：PC端支持屏幕任意区域取色，移动端支持拍摄画面取色；
5. **风格设计**：采用浮世绘经典藏青+朱红配色，搭配浅米色纸张质感，还原日式传统美学；
6. **异常兼容**：检测浏览器兼容性，对不支持 `EyeDropper API` 的浏览器给出友好提示。

## 🎨 设计风格
### 核心配色（浮世绘经典色）
| 元素         | 配色值                          | 设计说明                     |
|--------------|---------------------------------|------------------------------|
| 主按钮（取色）| 藏青渐变 `#2a4365` → `#1a365d`  | 还原浮世绘海浪/天空的藏青色 |
| 复制按钮     | 朱红渐变 `#e53e3e` → `#c53030`  | 复刻浮世绘服饰/印章的朱红色 |
| 背景/辅助    | 浅米色 `#f9f5f0`、浅木纹 `#e8dcc8` | 模拟浮世绘版画纸张质感      |
| 文字/点缀    | 柿红 `#8c2c22`、深棕 `#6b4c2d`  | 复古日式文字配色            |

### 交互细节
- 按钮hover时轻微上浮（`transform: translateY(-2px)`），搭配渐变加深，增强层次感；
- 低饱和度阴影+木纹边框，贴合浮世绘版画的纹理质感；
- 字体选用微软雅黑/黑体，适配日式复古风格。

## 🚀 使用方法

1. 点击「开始取色」按钮，选择屏幕任意像素完成取色；
2. 取色完成后，点击「复制颜色值」即可将十六进制颜色值复制到剪贴板。

### 核心代码片段
```javascript
// 唤起取色器核心逻辑
const eyeDropper = new EyeDropper();
const result = await eyeDropper.open();
const hexColor = result.sRGBHex; // 获取十六进制颜色值
```

## 📌 注意事项
1. 移动端取色需授予浏览器相机权限；
2. 部分浏览器在隐身模式下可能无法使用剪贴板复制功能；
3. 纯前端工具，无数据存储，所有操作均在本地完成，无需担心隐私泄露。
